Design Prompts

Visual guidelines for modern web design

  • Make it visually Appealing
  • Eye catching
  • Mobile Responsive
  • Add dynamic Animation
  • Make it look cool and beautiful
  • More modern, cool, and eye-catching palette
  • Use clean and modern typography
  • Ensure fast loading times
  • Make it Futuristic, sleek, and premium look
  • If using Next.js then follow component-wise code structure

Advanced Design Principles

  • Implement micro-interactions for user feedback
  • Use consistent spacing and alignment
  • Create depth with shadows and layers
  • Optimize for accessibility (WCAG compliance)
